Selected Work

Things I've shipped, and what they cost.

1

Thomson ERP /
Procurement Suite

// Live in production
VB.NETSQL ServerStored ProcsTriggersSSMS
The Problem
Procurement teams were stitching together purchase requisitions, vendor approvals, and goods receipts across spreadsheets and disconnected screens — every step was a chance to lose data, mis-route an approval, or ship the wrong PO.
The Solution
Built the full PR → PO → GRN lifecycle as a single integrated module inside Thomson ERP — supplier rating, CAPEX tracking, and inventory reconciliation all flowing through one source of truth, with optimized stored procedures handling the heavy reads.
The Impact
Procurement cycles became traceable, auditable, and fast. What used to require five tabs and a spreadsheet now runs in one workflow — and the SQL holds up under real load.
2

AI-Based
Parcel Delivery

// Real-time logistics
PythonGoogle Maps APIEmail AutomationCloud Deploy
The Problem
Parcel customers were chasing updates manually — calling support, refreshing dashboards, hoping. Delivery teams were sending status emails by hand. Visibility was a luxury, not a default.
The Solution
Built a real-time tracking system on Google Maps API with automated email pipelines that fire on every state change — dispatched, in-transit, out-for-delivery, delivered. Customers see their parcel move; ops never types another status update.
The Impact
Manual status work dropped to near-zero. Customer trust became something the system generated automatically instead of something support staff had to defend on the phone.
3

CAPEX & AMC
Tracking Module

// Asset lifecycle
VB.NETSQL ServerEmail TriggersReporting
The Problem
Capital expenses and Annual Maintenance Contracts were being tracked in scattered files with no alerts — renewal dates slipped past, asset depreciation went stale, and finance discovered problems weeks after they'd already cost money.
The Solution
Designed a centralized CAPEX + AMC tracker with automated email alerts on contract expiry, renewal windows, and maintenance due-dates — backed by SQL triggers that catch every state change the moment it happens.
The Impact
Renewals stopped slipping. Finance got a single dashboard instead of an inbox archaeology project, and maintenance teams could finally plan around real expiry data instead of memory.
